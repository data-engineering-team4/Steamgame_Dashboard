# 🎮 Steamgame_Dashboard
### Steam API를 활용한 Steam 게임 트렌드 및 유저 활동 분석
<br>

## ✔️ 프로젝트 개요
### 목적
약 140,000개가 넘는 Steam 게임의 상세 정보 및 리뷰 데이터, 유저 활동 데이터를 통해 게임 트렌드와 유저 활동에 대한 인사이트 도출
<br>

### 개발 기간
- 23.06.26 ~ 23.07.07 (2주)
